# cpp-scope-concepts
This project demonstrates the concept of:

## Global Variables
## Local Variables
## Variable Shadowing
## Scope Resolution Operator (::)
   in C++.

## Code Explanation

A global variable x is declared outside the main() function.

int x = 20;

A local variable with the same name is declared inside main().

int x = 50;

The local variable shadows the global variable.

To access the global variable, the Scope Resolution Operator is used:
::x

## Output
local x = 50
global x = 20
